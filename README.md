# Movie Quiz (Client-Side Web Development)

## Description
An interactive movie quiz created for a first-year Client-Side Web Development assignment. Built using HTML, CSS, and JavaScript, the quiz lets users answer multiple-choice questions and receive a final score.

## Features
- Multiple-choice quiz questions
- Score calculation and results display
- Instant feedback (optional if your quiz has it)
- Responsive layout (if applicable)

## Technologies
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone this repository.
2. Open `index.html` in a web browser.
